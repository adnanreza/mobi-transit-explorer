# Feature 011 - Navigation and Section Flow

Branch: `feature/navigation-section-flow`

## Goal

Make the app feel smooth and easy to explore.

## Scope

Wire the nav items to app sections.

## Behavior

Nav items should jump or scroll to Overview, Map, Opportunities, and Methodology.

## Requirements

- Active or hover state for nav items.
- Smooth scrolling if appropriate.
- Header remains clean and usable.
- Mobile nav remains simple.

## Tests

- Nav items render.
- Nav links target correct sections.
- Section ids exist.

## Acceptance Criteria

- App can be understood and explored quickly.
- Navigation feels intentional.
- No routing complexity is added unless needed.
