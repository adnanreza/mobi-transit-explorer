# Spec 044 — Copy sweep: no em dashes, simpler sentences

## Context

Owner direction: a full pass on the copy. No em dashes (the tic reads as AI-written), and simpler, easier English throughout, while keeping the professional, researcher-like voice.

## Rules applied

- Every em dash in user-facing prose was rewritten, not mechanically swapped: a period where the clause could stand alone, a colon where a list or definition followed, a comma for a light aside. Long multi-clause sentences got split.
- En dashes stay in numeric ranges ("2017–2026", "7:00–11:00", "0–100", "1–5×"). That is standard typography, not prose style.
- Two intentional glyph uses survive: the "—" placeholder for a null metric value in the overview adapter (standard no-data notation) and MapLibre's internal punctuation table (library code, never prose).
- Label separators in the map legend moved from em dash to colon ("Size: trip volume", "Missing: no Mobi dock within 1 km").

## Scope

~50 rewrites across the site and repo copy: src/App.tsx (hero + section descriptions), src/components/Methodology.tsx (22 instances, the bulk), story captions in src/components/story/storyContent.ts, src/components/PersonalRequests.tsx, src/components/flows/FlowsSection.tsx, src/components/story/WeatherModelBlock.tsx, Explorer/FilterPanel/RealMobiCharts/OpportunityTable/StationFinder/MapErrorBoundary/AppShell copy, map legend + aria labels in src/components/InteractiveMap.tsx, opportunity reason strings in src/data/index.ts, README.md (13 instances), and one string generated by the pipeline itself: the model card's station line in pipeline/train_model.py, with forecast.json regenerated and the freshness gate re-run so the committed artifact matches.

Code comments were left alone; they are not copy.

## Verification

- Grep-clean: zero em dashes in rendered prose across src, README.md, and index.html. The built bundles were audited too (comments strip in minification), which is how the pipeline-generated model-card string was caught.
- Every test-asserted phrase was preserved through the rewrites: 97 Vitest tests pass unchanged, plus typecheck and build. Pipeline side: model retrained (same deterministic outputs apart from the string), `make check-artifacts` PASS, 36 pytest tests pass.
